# Sistema de Cotizaciones

Aplicación desarrollada con Node.js y JavaScript para generar cotizaciones profesionales en PDF.

## Objetivo

Crear una aplicación moderna para:

- Gestión de clientes
- Gestión de cotizaciones
- Generación de PDF
- Historial de cotizaciones
- Búsqueda avanzada
- Compartir cotizaciones

Proyecto desarrollado por Jovanny con apoyo de ChatGPT.
